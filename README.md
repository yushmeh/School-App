**НЕКСУС АКАДЕМИЯ** — современная веб-система управления онлайн-школой, построенная на базе **FastAPI (Python 3.14)** с использованием архитектурного паттерна **Clean Architecture (3-Tier)**.

Интерфейс выполнен в виде интерактивного **Single Page Application (SPA)** в футуристичном стиле **Sci-Fi Dark Mode** с эффектами **Glassmorphism**.

## 🚀 Ключевые особенности и бизнес-логика

### 🏗 Архитектура
- **Clean Architecture / 3-Tier:** Строгое разделение на слои данных, бизнес-логики и представления.
- **Умная JSON-персистентность:** Данные автоматически сохраняются в `school_data.json` при каждом изменении. Связи между сущностями хранятся через ID, исключая дублирование и циклические ссылки.
- **Автосид при первом запуске:** Если база данных пуста, система автоматически заполняется тестовыми данными — 6 преподавателей, 7 студентов, 6 курсов с вложенными модулями, уроками и домашними заданиями.

### 📚 Модульная структура курсов
Курсы организованы в трёхуровневую иерархию:
```
Курс → Модуль → Урок → Домашнее задание
```
Каждый уровень можно создавать и редактировать через интерфейс или API.

### 📋 Система домашних заданий
- Преподаватель **выдаёт ДЗ** одному студенту или сразу всем зачисленным на курс.
- Преподаватель **выставляет оценку** (2–5) за каждое выполненное ДЗ с возможностью добавить комментарий.
- **Итоговый балл** по курсу рассчитывается динамически как среднее арифметическое всех оценённых ДЗ.
- **Правило аттестации:** если у студента оценено менее 3 ДЗ на момент завершения курса — в историю записывается **«Н/А»** (Не аттестован).

### 👩‍🏫 Управление преподавателями
- Статусы **«Активен»** / **«В отпуске»**: преподавателя в отпуске нельзя назначить на курс.
- **Защита от удаления:** преподавателя, ведущего активный курс, невозможно удалить из системы.

### 📊 Аналитика и отчёты
- **Отчёт по студенту** — активные курсы, прогресс по ДЗ, история завершённых курсов с финальными баллами и ASCII-визуализацией успеваемости `[████████░░] 80%`.
- **Отчёт по преподавателю** — количество активных курсов, средняя успеваемость студентов по всем ДЗ, список завершённых курсов с датами проведения.
- **Сводная аналитика** — общая статистика школы, топ-3 самых популярных курса, средний балл по школе.

### 📥 Экспорт в CSV
Любой отчёт можно выгрузить в CSV одним кликом (корректно открывается в Excel):
- Отчёт по студенту (курсы, прогресс, все оценки за ДЗ)
- Отчёт по преподавателю (метрики, активные и завершённые курсы)
- Отчёт по курсу (структура, список студентов и их успеваемость)

### ✅ Валидация
- Проверка корректности email при создании и редактировании студентов (на уровне Pydantic и в интерфейсе).
- Российская шкала оценок **2–5** с выпадающим списком подписей.
- Фильтрация курсов по статусу: все / активные / завершённые.

## 📂 Структура проекта

```text
school_app/
├── main.py           # FastAPI app, регистрация всех API-эндпоинтов
├── models.py         # Сущности и Pydantic-модели (Course, Module, Lesson, Homework, ...)
├── repository.py     # JSON-персистентность и менеджер состояния БД
├── services.py       # Бизнес-логика, аналитика, CSV-экспорт
├── seed.py           # Автозаполнение БД тестовыми данными при первом запуске
├── requirements.txt  # Зависимости проекта
├── school_data.json  # Локальная база данных (создаётся автоматически)
└── static/
    └── index.html    # SPA-интерфейс (Sci-Fi Dark Mode / Glassmorphism)
```

## 🗂 Схема данных

```
Student          Teacher          Course
├── id           ├── id           ├── id
├── name         ├── name         ├── title
├── email        ├── specializ.   ├── topic
└── history[]    └── status       ├── status
                                  ├── start_date / end_date
HomeworkSubmission                ├── teacher_id
├── id                            ├── enrolled_student_ids[]
├── homework_id                   └── modules[]
├── student_id                        └── Module
├── course_id                             ├── id / title / order
├── grade (2–5 | null)                    └── lessons[]
├── graded_by                                 └── Lesson
└── comment                                       ├── id / title / order
                                                  └── homeworks[]
                                                      └── Homework
                                                          ├── id / title
                                                          └── max_grade
```

## 🛠 Инструкция по запуску

Для работы необходим **Python 3.14**.

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Запустить сервер из папки school_app/
cd school_app
uvicorn main:app --reload --port 8000

# 3. Открыть в браузере
# http://localhost:8000
```

> При первом запуске с пустой базой данных система автоматически создаст тестовые данные через `seed.py`.

## 📡 API Эндпоинты

### Студенты
| Метод | URL | Описание |
| :---: | :--- | :--- |
| `GET` | `/api/students` | Список всех студентов |
| `POST` | `/api/students` | Создать студента |
| `PATCH` | `/api/students/{id}` | Обновить данные студента |
| `DELETE` | `/api/students/{id}` | Удалить студента |
| `GET` | `/api/students/{id}/card` | Карточка студента |
| `GET` | `/api/students/{id}/stats` | Статистика с ASCII-визуализацией |

### Преподаватели
| Метод | URL | Описание |
| :---: | :--- | :--- |
| `GET` | `/api/teachers` | Список всех преподавателей |
| `POST` | `/api/teachers` | Создать преподавателя |
| `PATCH` | `/api/teachers/{id}` | Обновить данные / сменить статус |
| `DELETE` | `/api/teachers/{id}` | Удалить преподавателя (защита от удаления с активным курсом) |
| `GET` | `/api/teachers/{id}/report` | Аналитический отчёт по преподавателю |

### Курсы
| Метод | URL | Описание |
| :---: | :--- | :--- |
| `GET` | `/api/courses` | Список всех курсов (с деталями) |
| `POST` | `/api/courses` | Создать курс |
| `PATCH` | `/api/courses/{id}` | Обновить курс |
| `GET` | `/api/courses/{id}/card` | Подробная карточка курса |
| `POST` | `/api/courses/{id}/complete` | Завершить курс |
| `POST` | `/api/courses/{id}/modules` | Добавить модуль |
| `POST` | `/api/courses/{id}/modules/{mid}/lessons` | Добавить урок |
| `POST` | `/api/courses/{id}/modules/{mid}/lessons/{lid}/homeworks` | Добавить ДЗ |
| `GET` | `/api/courses/{id}/submissions` | Все сдачи ДЗ по курсу |

### Действия
| Метод | URL | Описание |
| :---: | :--- | :--- |
| `POST` | `/api/enroll` | Зачислить студента на курс |
| `POST` | `/api/assign-teacher` | Назначить преподавателя на курс |
| `POST` | `/api/homework/assign` | Выдать ДЗ студенту или всем зачисленным |
| `POST` | `/api/homework/grade` | Выставить оценку за ДЗ |

| `GET` | `/api/export/teacher/{id}` | Экспорт отчёта по преподавателю в CSV |
| `GET` | `/api/export/course/{id}` | Экспорт отчёта по курсу в CSV |

### Служебные
| Метод | URL | Описание |
| :---: | :--- | :--- |
